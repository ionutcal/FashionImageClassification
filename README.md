# FashionImageClassification

Acest proiect vizeaza clasificarea automata a articelor vestimentare din setul de date Fashion MNIST utilizand tehnici de invatare automata. Obiectivul a fost dezvoltarea unui model capabil sa prezica corect categoria unei imagini.
Am ales metoda Random Forest, un algoritm de invatare supervizata bazat pe combinarea mai multor arbori de decizie independenti.
Etapele de implementare sunt: preprocesarea datelor, reducerea dimensionalitatii, antrenarea modelului si evaluarea.
Precizia modelului este de 0.8581. Acesta demonstreaza o capacitate solida de clasificare, desi performanta poate fi influentata de pierderea unor detalii fine in urma procesului PCA.
